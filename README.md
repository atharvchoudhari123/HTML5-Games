This is my unblocked game site. there is only a few games but i am planning to add more soon.

how to run:
1. Click code, codespaces, create codespace on main/master. let it load
2. Run the command `npm install` and after that completes, run `npm start`.
Hope you like it
Wanna add games? fork the repo and then create a new file in the games folder, then link it to the games.html page how it is done in the first few games. than create a pull request from your forked version and then i will check and merge.
